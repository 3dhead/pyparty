Returns a hexagonal skeleton (7 point honeycomb).

Center coordinates in pixels
d_pp : float
Particle-particle center distance.

n : int (2-4)

phi : float
Phase angle in degrees for orientation.
coordinates : tuple (n7)
Center coordinate (cx, cy) followed by hexagon verticies drawn a distance d_pp away.

From center coordnate (cx, cy), goes out a distance d, returns c1. Goes out a distance d dot 60 degrees, returns next point on hexagon.
