class pyparty.trait_types.metaparticle.MetaParticle(name='', color='', particle='')[source]¶

Stores a particle and metadata for use by ParticleManager.

This enforces initialization/validation of types without using Traits (to be light in memory). __slots__ prevents users from adding attributes accidentally. Using __slots__ and __setattr__ leads to some funky syntax as described:

http://stackoverflow.com/questions/19566419/can- setattr-can-be-defined-in-a-class-with-slots
